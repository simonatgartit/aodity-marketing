Funktioner

Allt du behöver för masterdata

Från ett enkelt Excel-tillägg till ett fullständigt API, byggt för team som hanterar strukturerad data. Data når ditt BI-verktyg eller data warehouse via REST API med API-nyckel.

📊

Publicera från Excel

Ladda upp data direkt från Excel-tillägget med ett knapptryck. Ingen export, ingen manuell hantering.

🔀

Versionskontroll

Varje publicering skapar en ny version. Bläddra i historiken och återställ ett tidigare tillstånd när som helst.

🔐

Rollbaserad access

Manager, editor och viewer. Styr vem som får publicera, redigera eller enbart läsa. Allt tenant-isolerat.

🔗

Datavalidering och relationer

Definiera regler direkt i schemat. Kräv att värden matchar en lista, följer ett format eller refererar till en annan entity. Fungerar i både Excel och webb-editorn.

Hur det fungerar

Tre steg till strukturerad data

Kom igång på minuter. Ingen infrastruktur att sätta upp, ingen kod att skriva.

1

Definiera schema

Skapa ett entity-schema i admin-portalen och definiera fält med namn och typ. Lägg till valideringsregler som enum-värden, format eller kopplingar till andra entities.

2

Publicera från Excel

Öppna Aodity-tillägget i Excel, välj entity och tryck Publicera. Data valideras mot schemat och lagras som en ny version.

3

Hämta via API

Integrera med ditt BI-verktyg via REST API. Alltid senaste versionen, eller välj en specifik version för reproducerbara rapporter.
